# knex-boilerplate
Boilerplate for Knex.js with Express

Boilerplate for projects with:
- Knex
- Express
- SQLite3
- Handlebars
- Jest
- Nodemon

Contains:
- migrations folder
- seeds folder
- tests folder
- views folder: partials folder and layouts folder 
- index.js 
- server.js
- routes folder: routes.js
- package.json
- knexfile
